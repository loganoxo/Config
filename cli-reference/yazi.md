# yazi

https://yazi-rs.github.io/docs/quick-start

## 一、[安装](https://yazi-rs.github.io/docs/installation/)

```shell
# 安装
brew install yazi ffmpeg sevenzip jq fd ripgrep fzf zoxide

# 可选
brew install imagemagick poppler
# imagemagick : 用于 SVG、字体、HEIC 和 JPEG XL 预览
# poppler : 用于 PDF 预览
```

## 二 、[基础按键](https://yazi-rs.github.io/docs/quick-start)

### 1、系统

| **按键** |        **作用**        |
| :----- | :------------------: |
| `q`    | `退出`,并`切换`外部终端的当前目录  |
| `Q`    | `退出`,`不切换`外部终端的当前目录  |
| `~`    |       打开`帮助菜单`       |
| `;`    |    运行 `shell` 命令     |
| `:`    | 运行 `shell` 命令,阻塞直到完成 |
| `!`    |  在当前目录打开shell(自定义)   |

### 2、导航

- 默认

| **按键**    | -------------------- **作用** -------------------- |
| :-------- | :----------------------------------------------: |
| `k/j`     |                    `上/下` 移动光标                    |
| `l`       |                   `进入`光标选择的目录                    |
| `h`       |                 `离开`当前目录并进入其父目录                  |
| `gg`      |                    将光标移至`顶部`                     |
| `G`       |                    将光标移动到`底部`                    |
| `z`       |                使用 `zoxide` 跳转到目录                 |
| `Z`(大写)   |               使用 `fzf` 跳转到目录或显示文件                |
| `H`       |      使用 `zoxide` 或者 `fzf` 跳转后 可以`回到跳转前`的目录       |
| `L`       |                  `H` 回去 之后 再回来                   |

- 自定义  

| **按键** | -------------------- **作用** -------------------- |
| :----- | :----------------------------------------------: |
| `gt`   |                    cd ~/Temp                     |
| `gT`   |                  cd ~/TempCode                   |
| `gw`   |                   cd ~/Desktop                   |
| `gd`   |                    cd ~/Data                     |
| `gc`   |                 cd ~/Data/Config                 |
| `gr`   |          cd ~/Data/Config/cli-reference          |
| `gh`   |                       cd ~                       |
| `gg`   |                   进入git仓库的根目录                    |

### 3、预览窗口

| **按键**    | -------------------- **作用** -------------------- |
| :-------- | :----------------------------------------------: |
| `K/J`(大写) |            在 `预览窗口` 中, 向上/下 滚动 5 个单位             |
| `=` + `h` |                    `隐藏` 预览 窗口                    |
| `=` + `m` |                   `最大化` 预览 窗口                    |
| `=` + `p` |               使用 mac系统 的快速预览 查看文件                |
|           |                                                  |

### 4、文件(夹)选择

| **按键**    | -------------------- **作用** -------------------- |
| :-------- | :----------------------------------------------: |
| `<Space>` |              `切换/选择` 悬停的文件/目录; 可以多选              |
| `v`       |                 进入 类似 vim 的选择模式                  |
| `<C-a>`   |                 `选择` 当前目录下的 所有文件                 |
| `<C-r`>   |     `逆向选择`  (把当前目录下的, 已选择的切换为未选择,未选择的切换为 选择)     |
| `Esc`     |                       取消选择                       |

### 5、文件(夹)操作(交互)

| **按键**                   | -------------------- **作用** -------------------- |
| :----------------------- | :----------------------------------------------: |
| `o` 或者 `Enter`           |                 `打开` 已选择的文件或文件夹                  |
| `O`(大写) 或者 `Shift-Enter` |     `以交互方式` 打开已选择的文件或文件夹 (Reveal=在finder中打开)     |
| `Tab`                    |                     显示`文件信息`                     |
| `y`                      |                  `复制` 选中的文件或文件夹                  |
| `x`                      |                  `剪切` 选中的文件或文件夹                  |
| `p`                      |                  `粘贴` 复制的文件或文件夹                  |
| `P` (大写)                 |              `粘贴并覆盖`, 如果目标位置存在同名的话               |
| `Y` / `X` (都大写)          |              `取消` 复制/剪切 的文件或文件夹的状态               |
| `d`                      |                    `删除,放入废纸篓`                    |
| `D`                      |                      `永久删除`                      |
| `a`                      |           `新建` 文件/文件夹 (以 `/` 结尾就是文件夹)            |
| `r`                      |          `重命名` 选中的文件/文件夹 (可多选,调用vim修改)           |
| `.`                      |                  `显示` 隐藏文件 / 切换                  |
| `-`                      |        `创建` 已选择复制的文件或文件夹的 `软链接` (`绝对路径`)         |
| `_`                      |        `创建` 已选择复制的文件或文件夹的 `软链接` (`相对路径`)         |
| `<CTRL>` + `-`           |     `创建` 已选择复制的文件或文件夹的 `硬链接` (好像按键被占用了,不能用)      |

- 自定义
    - `<p>`   智能粘贴; 选中目录后,直接把复制的内容粘贴到选中的文件夹下,不用进入文件夹了;如果要粘贴到当前目录,则需选中当前目录中的文件

### 6、复制路径

| **按键**    | -------------------- **作用** -------------------- |
| :-------- | :----------------------------------------------: |
| `c` + `c` |                `复制` 文件/文件夹 的 绝对路径                |
| `c` + `d` |           `复制` 文件/文件夹 的 `所处的目录的` 绝对路径            |
| `c` + `f` |                `复制` 文件/文件夹 的 `名字`                |
| `c` + `n` |             `复制` 文件/文件夹 的 `不带扩展名的名字`             |

### 7、搜索

| **按键**         |      -------------------- **作用** --------------------      |
| :------------- | :--------------------------------------------------------: |
| `f`            | `过滤` 文件/文件夹 (按 `Enter` 后退出搜索框, 会保持搜索内容, 然后可以按 `ESC` 取消搜索 ) |
| `/`            |                 查找 在光标之`后` 的文件或文件夹(类似vim)                  |
| `?`            |                 查找 在光标之`前` 的文件或文件夹(类似vim)                  |
| `n`            |                       找 下 一个(类似vim)                        |
| `N`            |                       找 上 一个(类似vim)                        |
| `s`            |                    使用 `fd` 按名称搜索 文件或文件夹                    |
| `S` (大写)       |                    使用 `ripgrep` 按内容搜索文件                    |
| `C-s` 或者 `ESC` |                        `取消` 正在进行的搜索                        |
| `F`            |                  使用插件 智能搜索; 连续搜索; 提交后打开文件                  |

### 8、排序

| **按键**    | -------------------- **作用** -------------------- |
| :-------- | :----------------------------------------------: |
| `,` + `a` |                   按 `字母顺序` 排序                    |
| `,` + `A` |                按 `字母顺序` 排序 (`倒序`)                |
| `,` + `m` |                   按 `修改时间` 排序                    |
| `,` + `M` |                按 `修改时间` 排序 (`倒序`)                |
| `,` + `b` |                   按 `创建时间` 排序                    |
| `,` + `B` |                按 `创建时间` 排序 (`倒序`)                |
| `,` + `e` |                   按 `文件扩展名` 排序                   |
| `,` + `E` |               按 `文件扩展名` 排序 (`倒序`)                |
| `,` + `n` |                     `自然` 排序                      |
| `,` + `N` |                  `自然` 排序 (`倒序`)                  |
| `,` + `s` |                    按 `大小` 排序                     |
| `,` + `S` |                 按 `大小` 排序 (`倒序`)                 |
| `,` + `r` |                     `随机` 排序                      |

### 9、多标签页

| **按键**       | -------------------- **作用** -------------------- |
| :----------- | :----------------------------------------------: |
| `t`          |                 使用 当前目录 创建一个新标签页                 |
| `1,2,...,9`  |                `数字键` `切换` 到对应的标签页                |
| `[`          |                  切换到 `上一个` 标签页                   |
| `]`          |                  切换到 `下一个` 标签页                   |
| `{`          |               `交换` 当前标签页与`上`一个标签页                |
| `}`          |              `交换` 当前标签页与 `下` 一个标签页               |
| `Ctrl` + `c` |                    关闭当前 `标签页`                    |

- 自定义
- 按 `t`;  如果当前选中的是文件夹, 按 t 创建新的标签页时,会在新标签页中直接进入当前选中的文件夹

### 10、任务管理器

- 长时间的任务,如复制大文件等; 或者失败的任务,会在这里

| **按键**                      | -------------------- **作用** -------------------- |
| :-------------------------- | :----------------------------------------------: |
| `w`                         |                     显示任务管理器                      |
| `ESC` 或 `C-c` 或 `C-[` 或 `w` |                    退出任务管理器窗口                     |
| `Enter`                     |                    查看某个任务的详情                     |
| `x`                         |                      取消某个任务                      |
